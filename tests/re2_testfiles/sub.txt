Tests of substitution
=====================

This first test is just looking to replace things between parentheses
with an empty string.


    >>> import re2 as re
    >>> import hashlib
    >>> import gzip
    >>> import os
    >>> import doctest
    >>> path = os.path.abspath(os.getcwd())
    >>> path = os.path.join(path, 'tests', 're2_testfiles', 'wikipages.xml.gz')
    >>> data = gzip.open(path).read()
    >>> print(hashlib.md5(re.sub('\(.*?\)', '', data).encode('utf-8')).hexdigest())
    b7a469f55ab76cd5887c81dbb0cfe6d3
